# 模糊综合评价
模糊评价法把主观的评价通过量化的处理变成综合得分的方法，主观的评价比如说“很好，好，差”这种的。

隶属度：隶属于某个模糊评价的度。

## 模糊统计法

例如评价某个同学在学校的综合表现，有优、良、差三个等级，这里就是模糊评价，数字代表评价的人数

|          | 优   | 良   | 差   |
| -- | :--- | ---- | ---- |
| 成绩     | 20   | 30 | 0 |
| 学术研究 | 16   | 25 | 9 |
| 志愿服务 | 9    | 17 | 24 |
| 责任感   | 18   | 13 | 19 |

然后算出对应的评价等级占当前指标的比例得到另一张表，隶属度矩阵。

| 指标       | 非常满意 | 满意 | 一般 | 不满意 |
| ---------- | -------- | ---- | ---- | ------ |
| 服务效率   | 0.25     | 0.50 | 0.20 | 0.05   |
| 服务态度   | 0.30     | 0.45 | 0.20 | 0.05   |
| 信息透明度 | 0.15     | 0.50 | 0.25 | 0.10   |
| 使用便利性 | 0.20     | 0.55 | 0.20 | 0.05   |

然后通过各种定权的方式得到权重，用权重来与隶属度矩阵相乘，可以得到最终的一个评价，例如

| 等级     | 得分 |
| -------- | ---- |
| 非常满意 | 0.23 |
| 满意     | 0.49 |
| 一般     | 0.22 |
| 不满意   | 0.06 |

取最高的得分就是最后得到的等级
