# 数组（Array）
长度固定，存储在栈上

**数组创建**
```
let a: [i32; 5] = [1, 2, 3, 4, 5];
[类型；长度]
```

还可以使用下面的语法初始化一个某个值重复出现 N 次的数组：
```
let a = [3; 5];
可以左边使用定义，右边使用默认值的形式
```
a 数组包含 5 个元素，这些元素的初始化值为 3，这种语法跟数组类型的声明语法其实是保持一致的：[3; 5] 和 [类型; 长度]。

数组的长度也是类型的一部分，不同长度的数组类型不同

数组内的元素类型必须是一致的

**数组访问**
通过索引的方式访问，也是从0开始索引

**数组元素为非基本类型**

不能使用上面的copy方法

## 数组切片
```
let a: [i32; 5] = [1, 2, 3, 4, 5];
let slice: &[i32] = &a[1..3];
assert_eq!(slice, &[2, 3]);
```
声明切片的时候使用引用加上数组类型的创建方法
